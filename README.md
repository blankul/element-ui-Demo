# 这是一个可以快速上手（element-ui + vue) 的模板

> A Vue&element-ui.js project

## Build Setup

``` bash
# install dependencies  // 安装开发依赖
npm install

# serve with hot reload at localhost:666 // 启动热加载的服务器：localhost:666
npm run dev

# build for production with minification // 生成产品
npm run build
```
